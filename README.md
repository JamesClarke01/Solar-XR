# Mars XR

## Installation
The following addons must be installed from the asset library:

| Name                       | Version | Installation Path     |
| -------------------------- | ------- | --------------------- |
| Godot XR Tools for Godot 4 | 4.3.3   | addons/godot-xr-tools |
| Debug Draw 3D (4.1.4+)     | 1.4.5   | addons/debug_draw_3d  |
